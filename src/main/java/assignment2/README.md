# 동전 조합 계산기
주어진 동전들로 특정 금액을 만들 수 있는 모든 조합의 수를 계산하는 프로그램

## 기능
- 여러 종류의 동전으로 목표 금액을 만드는 조합 수 계산
- 각 동전은 무제한으로 사용 가능
- 동적 계획법을 활용

## 예시
```aidl
입력: coins = [1,2,3], sum = 4
출력: 4
조합: {1,1,1,1}, {1,1,2}, {2,2}, {1,3}
입력: coins = [2,5,3,6], sum = 10
출력: 5
조합: {2,2,2,2,2}, {2,2,3,3}, {2,2,6}, {2,3,5}, {5,5}
```
